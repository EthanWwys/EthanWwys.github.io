组合数学 暴力枚举

Posted by 王熠山 on Friday, October 4, 2024

题目描述

有 $n$ 根木棒，现在从中选 $4$ 根，想要组成一个正三角形，问有几种选法？

答案对 $10^9+7$ 取模。

输入格式

第一行一个整数 $n$。

第二行往下 $n$ 行，每行 $1$ 个整数，第 $i$ 个整数 $a_i$ 代表第 $i$ 根木棒的长度。

输出格式

一行一个整数代表答案。

样例 #1

样例输入 #1

4 
1
1
2
2

样例输出 #1

1

提示

数据规模与约定

  • 对于 $30%$ 的数据，保证 $n \le 5 \times 10^3$。
  • 对于 $100%$ 的数据，保证 $1 \leq n \le 10^5$，$1 \le a_i \le 5 \times 10^3$。

关于标题：因为一些不可抗力的原因，名称进行了更改。深表歉意。

题目分析

由四根木棒组成一个等边三角形，通过简单数学实验可知必然会有两根木棒$l_1,l_2$长度相等（最长的两根）剩下的两根长度之和等于$l_1,l_2$的长度，例如2,2,1,1。 因此，解题思路可以是从长度由大到小枚举，考察最长边为$l_i$的两根木棒可以和其他小木棒构成多少个等边三角形。同时注意到 $1 \leq n \le 10^5$，$n^2$的时间复杂度可以通过。

具体实现

预处理，构建数组

下标为i的位置存储长度为i的木棒的数量

循环部分

外层循环

从最长的木棒$l_i$开始枚举（因为最长的木棒决定了等边三角形的边长是多少，决定了两个短木棒的组合），长度为$l_i$木棒数如果超过两个，则具有组成等边三角形最基础的条件。接下来选择2个长度为$l_i$的木棒，数量为k = straw[i] * (straw[i] - 1)/2;个。

内层循环

内层循环用于计算在已经选出两个特定的长木棒的情况下，有多少个短木棒的组合。也就是两个短木棒的长度之和要等于长木棒的长度，问题自然就转化成将长木棒分解为两个正整数的不定方程（把6拆成15,24,33），并且计算每一个分解结果有多少种组合。

每一个分解结果组合数自然是两个长度对应的木棒数的乘积straw[j] * straw[i-j];然而这里需要注意，如果长木棒的长度是偶数，处理内层循环的时候会遇到刚好两个短木棒长度为长木棒一半的情况，此时这种情况的数量不能通过straw[j] * straw[i-j];计算，需要通过单独判断来处理，因此，选择在循环到$l_i/2$​之前结束循环，通过判断i是否为偶数在额外计算两个短木棒长度为长木棒一半的组合数。

对temp累加

回到外层循环

and += k * temp结果加上长木棒组合数与短木棒组合数的乘积，取模之后就是正确答案咯！！

AC代码

#include<bits/stdc++.h>
using namespace std;
const int N = 1e5 + 10;
const int M = 1e9 + 7;
int main(void)
{
    long long straw[N] = {0};
    long long n, length, max = 0, ans = 0, k;
    cin>>n;
    for(int i = 1; i<=n; i++)
    {
        cin>>length;
        straw[length]++;
        if(length > max)    max = length;
    }//构建数组，下标为i的位置存储长度为i的木棒的数量

    for(int i = max; i>=2; i--)
    {
        long long temp = 0;
        k = straw[i] * (straw[i] - 1)/2;
        if(straw[i]>=2)
        {
            for(int j = i; j > i/2; j--)
                temp += straw[j] * straw[i-j];
            if(!(i%2))
                temp += straw[i/2]*(straw[i/2]-1)/2;
        }
        ans += temp * k;
    }
    cout<<ans%M;
}
